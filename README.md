# 📚 Library Management System (C++)

A simple **console-based Library Management System** developed using **C++**.  
This project is designed to manage basic library operations through a menu-driven interface and demonstrates the implementation of fundamental programming concepts.

The system allows users to add books, display book records, search books, borrow books, return books, and exit the application.

---

## 🚀 Features

- 📖 Add new books
- 📋 Display all books
- 🔍 Search books by details
- 📥 Borrow books
- 📤 Return borrowed books
- ❌ Exit the system
- 🖥️ User-friendly command-line interface

## 💻 How to Run the Project

### Prerequisites

Make sure you have:

- C++ compiler installed (GCC/MinGW)
- Visual Studio Code or any C++ IDE

---

### Steps

### 1. Clone the repository

```bash
git clone https://github.com/your-username/cpp-library-management-system.git
```

### 2. Open the project directory

```bash
cd cpp-library-management-system
```

### 3. Compile the program

```bash
g++ src/main.cpp -o library
```

### 4. Run the program

#### Windows

```bash
library.exe
```

#### Linux / Mac

```bash
./library
```
